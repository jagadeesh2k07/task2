## What's Built

A responsive Login and Registration UI built with HTML, CSS and JavaScript — frontend only.

**Login**
- Email & password validation
- Show/hide password toggle
- Google & GitHub buttons (open provider pages)
- Redirects to portfolio on successful login

**Register**
- Name, email, phone fields with validation
- Live email availability check (AJAX simulation)
- Password strength bar (rule-based)
- Confirm password match indicator
- Terms checkbox
